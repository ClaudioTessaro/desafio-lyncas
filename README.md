# Api de contas a pagar

## Descrição

Api de contas a pagar, onde é possível cadastrar, listar, editar e excluir contas a pagar.

## Utilização

Para utilizar a api, é necessário ter o docker e docker-compose instalados.

Para rodar a aplicação, execute o seguinte comando:

```bash
docker-compose up
```
Para rodar os testes de unidades
    
```bash
mvn test
```

## Endpoints da API

Você pode acessar a documentação da API visitando a seguinte URL:

```
http://localhost:8080/swagger-ui.html
```

OU Você pode acessar a documentação da API pelo arquivo de coleção do postman no diretório raiz do projeto com o nome `Api de contas a pagar.postman_collection.json.postman_collection.json)`
